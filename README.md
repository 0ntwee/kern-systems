## 📖 О ПРОЕКТЕ

**KERN SYSTEMS** — это исследовательская инициатива по созданию устойчивых автономных платформ. 
Фокус текущего этапа (**Phase 0**): разработка архитектуры управления, протоколов безопасности и прототипа мобильной платформы **«Портер»** (дипломный проект).

### 🎯 Миссия
Инженерная защита жизни и инфраструктуры там, где человек уязвим. Технологии как инструмент устойчивости, а не разрушения.

### ⚠️ ЭТИЧЕСКИЙ УСТАВ (Кратко)
1. **Запрет милитаризации:** Технологии не используются для причинения вреда человеку.
2. **Протокол «Свидетель»:** Полная прозрачность логов инцидентов.
3. **Открытое наследие:** Код ядра открыт для аудита.
4. **Право на контроль:** Пользователь всегда может отключить автоматику.

## 🏗️ АРХИТЕКТУРА

Система построена по микросервисному принципу для обеспечения отказоустойчивости.

```
┌─────────────────────────────────────┐
│  KERN ORCHESTRATOR (Web UI :8080)   │ ◄── Пользователь
└─────────────────────────────────────┘
           │ ▲
           ▼ │
┌─────────────────────────────────────┐
│  POLAR COMMANDER (API :5000)        │ ◄── Голос / Мониторинг
└─────────────────────────────────────┘
           │
           ▼
┌─────────────────────────────────────┐
│  MAP SERVICE (Planning :5002)       │ ◄── Навигация
└─────────────────────────────────────┘
           │
           ▼
┌─────────────────────────────────────┐
│  HARDWARE LAYER (ESP32 / Arduino)   │ ◄── «Спинной мозг»
└─────────────────────────────────────┘
```

### 🔑 Ключевые компоненты

| Сервис | Порт | Описание | Статус |
|--------|------|----------|--------|
| `kern_orchestrator.py` | 8080 | Центр управления, веб-интерфейс | ✅ Active |
| `polar_commander.py` | 5000 | API команд, голосовой ввод (RHVoice) | ✅ Active |
| `map_service.py` | 5002 | Картография, планирование путей | ✅ Active |
| `night_cycle.py` | - | Фоновые задачи по расписанию | ✅ Active |
| `protector_v2.c` | - | Защита исполняемого кода (Anti-Debug) | 🧪 Test |

## 🚀 БЫСТРЫЙ СТАРТ (LOCAL)

Для развёртывания локального ядра управления (Linux/Windows):

```bash
# 1. Клонирование
git clone https://github.com/KERN-SYSTEMS/kern-orchestrator.git
cd kern-orchestrator

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Запуск микросервисов (в разных терминалах)
python polar_commander.py          # Порт 5000
python map_service.py              # Порт 5002
python kern_orchestrator.py        # Порт 8080

# 4. Проверка
curl http://localhost:5000/system/check
```

> **Примечание:** Для работы голосового модуля требуется установленный `RHVoice` и `spd-say`.

## 🔐 БЕЗОПАСНОСТЬ И ЗАЩИТА

Проект разрабатывается в рамках специальности **«Обеспечение информационной безопасности автоматизированных систем»**.

*   **Три Ключа:** Аппаратная аутентификация + Криптоподпись + Поведенческий мониторинг.
*   **Защита кода:** Реализованы механизмы anti-debug через `/proc/self/status` и XOR-шифрование сегментов памяти (`protector_v2.c`).
*   **Протокол «Свидетель»:** Логи критических событий не подлежат удалению.
