## 📖 О ПРОЕКТЕ

**KERN SYSTEMS** — это исследовательская инициатива по созданию устойчивых автономных платформ. 
Фокус текущего этапа (**Phase 0**): разработка архитектуры управления, протоколов безопасности и прототипа мобильной платформы **«Портер»** (дипломный проект).

### 🎯 Миссия
Инженерная защита жизни и инфраструктуры там, где человек уязвим. Технологии как инструмент устойчивости, а не разрушения.

### ⚠️ ЭТИЧЕСКИЙ УСТАВ
1. **Запрет милитаризации:** Технологии не используются для причинения вреда человеку.
2. **Протокол «Свидетель»:** Полная прозрачность логов инцидентов.
3. **Открытое наследие:** Код ядра открыт для аудита.
4. **Право на контроль:** Пользователь всегда может отключить автоматику.
